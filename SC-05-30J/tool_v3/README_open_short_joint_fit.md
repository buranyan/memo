# 開放・短絡同時フィッティング版

今回の5 m・2芯ケーブルについて、同じ治具と校正条件で取得した

- 遠端開放
- 遠端短絡

のCSVを同時に使用して、差動線路定数を推定します。

## 実行

```bat
py mvvs_ul1007_open_short_joint_fit.py ^
  Zm_2C_normal_20260712.csv ^
  Zm_2C_normal_short_20260712.csv ^
  --length-m 5 ^
  --output-dir fit_5m_open_short
```

## 同定する値

- 差動特性インピーダンス Z0
- 速度係数 VF
- 周波数依存損失
- 有効誘電損失
- 共通測定治具の直列R・L
- 遠端短絡部のR・L

ケーブルの直流ループ抵抗は、既定の導体抵抗
`0.0378 Ω/m/芯` と長さから計算します。

## 開放側の下限周波数

開放測定は低周波で非常に高いインピーダンスとなり、
測定器入力抵抗や測定上限の影響を受けやすいため、
既定では50 kHz以上をフィットに使用します。

短絡測定は1 kHz以上を使用します。

## 出力

- `fit_open_short_summary.json`
- `fitted_5m_open_config.json`
- `fitted_5m_short_config.json`
- 開放・短絡の比較CSV
- 開放・短絡の振幅・位相グラフ

## 開放端容量について

既定では理想開放（0 pF）です。

開放端容量まで同時に自由化すると、速度係数との相関が強く、
一意に分離しにくくなります。小さな開放端容量も検討する場合は、

```bat
--fit-open-capacitance
```

を追加してください。探索範囲は0～15 pFです。
